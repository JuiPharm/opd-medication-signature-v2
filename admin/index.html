เวลา HN ผู้ทำรายการ ผู้รับยา ลายเซ็น

ส่งผู้ป่วยไปเครื่องเซ็น

ข้อมูลจะถูกส่งเฉพาะภายใน account นี้

สถานะคิวรอเซ็น

ดูได้ว่า case ค้างอยู่ตรงไหน: PENDING = รอเครื่องเซ็น, SIGNING = เปิดหน้าเซ็นแล้ว ถ้าหน้าหายให้กดส่งซ้ำ, COMPLETED = เซ็นสำเร็จ, CANCELLED = ยกเลิก

อัปเดตล่าสุด สถานะ HN / VN ชื่อ-นามสกุล ผู้ส่ง เครื่องที่รับคิว จัดการ

Staff ID ชื่อ-นามสกุล สิทธิ์ (Role) จัดการ
